# chat-script-brsp

### install
```
npm install --save chat-script-brsp
```


### usage

```
import React, { Component } from 'react'

import { ChatComponent } from 'chat-script-brsp';

const MyComponent = () => {
  return (
    <div><ChatComponent /></div>
  )
}
export default MyComponent;

```

### keywords

React, Typescript, css
